   - Guard: `if (!browser || AUTH_MODE !== 'amplify') return`
   - Calls `const result = await fetchShelves()`
   - Updates `appState` via `.update()` spreading `shelves: result`
   - Wraps in try/catch calling `reportSyncError('hydrateShelves', error)` on failure
5. Call `void hydrateShelves()` at the end of `hydrateCurationState()` (after the existing `appState.update` on ~line 107), so shelves load alongside curation workspace data
6. Export a derived store after line 183: `export const shelves = derived(appState, ($state) => $state.shelves)`
7. Export `createShelf(title: string, voiceIds: string[] = [])`:
   - Guard: `if (!curationWriteEnabled()) return`; also return early if `!title.trim()`
   - Build a temporary `CurationShelf`: `id` = `crypto.randomUUID()` (throwaway — the update path in `saveShelf` will fail because no DynamoDB record exists with this ID, triggering the create fallback which omits `id` so Amplify auto-generates the real one), `key` = `title.trim().toLowerCase().replace(/\s+/g, '-')`, `description: ''`, `voiceIds`, `published: false`, `updatedAt: new Date().toISOString()`
   - Optimistic: append placeholder shelf to `appState.shelves` immediately via `appState.update()`
   - Call `await saveShelfRemote(shelf)` then `await hydrateShelves()` to replace the placeholder with the real Amplify record (which has the DynamoDB-assigned ID)
   - On error: `reportSyncError('createShelf', error)` then `void hydrateShelves()` to revert
8. Export `updateShelf(shelfId: string, patch: Partial<Pick<CurationShelf, 'title' | 'description' | 'voiceIds' | 'published'>>)`:
   - Guard: `if (!curationWriteEnabled()) return`
   - Find existing shelf from `getAppStateSnapshot().shelves`; return early if not found
   - Build merged shelf: spread existing, spread patch, set `updatedAt: new Date().toISOString()`
   - Optimistic: replace matching shelf in `appState.shelves` via `appState.update()`
   - Call `await saveShelfRemote(mergedShelf)` (hits update path since ID exists in DynamoDB)
   - On error: `reportSyncError('updateShelf', error)` then `void hydrateShelves()` to revert
9. Export `deleteShelf(shelfId: string)`:
   - Guard: `if (!curationWriteEnabled()) return`
   - Optimistic: filter out shelf with matching `id` from `appState.shelves` via `appState.update()`
   - Call `await deleteShelfRemote(shelfId)`
   - On error: `reportSyncError('deleteShelf', error)` then `void hydrateShelves()` to revert
